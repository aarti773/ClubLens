# ClubLens

AI-powered Event & Media Management Platform for clubs and societies.

## Overview

ClubLens is a MERN-based web platform that helps clubs organize event-wise media, upload photos, search media intelligently, interact with photos, and discover personal photos using facial recognition.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Cloudinary
- face-api.js

## Key Features

- User authentication and role-based access
- Event creation and event-wise albums
- Cloudinary media upload
- Bulk image upload with preview
- Public and private media access
- Likes, comments, favourites, and sharing
- User tagging with notifications
- AI-generated tags
- Advanced media search
- Facial-recognition based personalized gallery
- Watermarked image downloads
- Admin role management dashboard
